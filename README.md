# 🏥 Hospital Management System — MERN Stack

A full-stack Hospital Management System built using **MongoDB, Express.js, React.js, and Node.js**.  
It supports **role-based access** for Admin, Doctor & Patient to manage appointments, medical data, users and dashboards efficiently.

---

## 🚀 Features

### 🔐 Authentication & Authorization
- JWT-based secure authentication
- Role-based access control (Admin / Doctor / Patient)

### 👨‍⚕️ Admin Panel
- Manage doctors & patients
- Approve doctor registration requests
- Dashboard insights

### 👨‍⚕️ Doctor Panel
- View assigned patients
- Manage appointments & medical records
- Update treatment status

### 🧑‍⚕️ Patient Panel
- Book/cancel appointments
- View prescription & medical history
- Manage profile

### 💡 Additional Features
- Responsive UI with React.js
- Optimized API calls for better performance
- Secure routing & form validation

---

## 🛠️ Tech Stack

| Layer | Technology |
|------|------------|
| Frontend | React.js, React Router, Axios, Bootstrap |
| Backend | Node.js, Express.js |
| Database | MongoDB, Mongoose |
| Authentication | JWT, Bcrypt |
